# 🚚 Shipment Data Analysis Tool

A logistics intelligence tool that lets managers upload CSV shipment 
data and ask questions in plain English — getting instant charts and 
tables back.

## 🔗 Live Demo
https://logistic-analysis-prototype.vercel.app/

## 🎯 Features
- Upload any CSV with shipment data
- Ask questions in plain English (e.g. "which routes had most delays?")
- Get instant bar charts, line charts, or data tables
- Sample CSV included for quick testing

## 🛠 Tech Stack
- React + TypeScript + Vite
- Shadcn/UI for components
- Claude API for natural language processing
- Recharts for data visualization

## 🚀 Running Locally
npm i
npm run dev

## 📋 Sample Questions to Try
- "Which routes had the most delays last month?"
- "Show total shipments by city"
- "What was the average delivery time?"
- "Which carrier performed best?"
